# COMP551: Applied Machine Learning

We use this repository to share the code for some of the topics that we cover in the McGill COMP551 class. Note that the recent updates in 2025 include the use of LLM tools to generate comments or parts of the code.

Here are the list of topics covered:

Maximum Likelihood and Bayesian Inference ([Gaussian](Gaussian.ipynb), [MLE_BayesianInference.ipynb](MLE_BayesianInference.ipynb))

Linear Classification and Regression
    [Naive Bayes] ([NaiveBayes.ipynb](NaiveBayes.ipynb))
    Linear Regression ([LinearRegression.ipynb](LinearRegression.ipynb))
    Logistic Regression ([LogisticRegression.ipynb](LogisticRegression.ipynb))
    Perceptron and Support Vector Machines ([Perceptron_and_LinearSVM.ipynb](Perceptron_and_LinearSVM.ipynb))

Training and Evaluation
    Gradient Descent ([GradientDescent.ipynb](GradientDescent.ipynb))
    Regularization ([Regularization.ipynb](Regularization.ipynb))
    Generalization and Model Selection ([ModelSelection.ipynb](ModelSelection.ipynb), [CurseOfDimensionality.ipynb](CurseOfDimensionality.ipynb))

Deep Networks
    Multilayer Perceptron ([MLP.ipynb](MLP.ipynb))
    Automatic Differentiation ([MLP.ipynb](MLP.ipynb))
    Deep Learning with Image Data ([ConvNets.ipynb](ConvNets.ipynb))
    Deep Learning with Sequential Data ([RNN.ipynb](RNN.ipynb))

Nonparametric Methods
    Nearest Neighbours ([KNN.ipynb](KNN.ipynb))
    Decision Trees and Random Forests ([DecisionTree.ipynb](DecisionTree.ipynb))

Unsupervised Learning
    Clustering ([KMeansClustering.ipynb](KMeansClustering.ipynb), [EMforGaussianMixture.ipynb](EMforGaussianMixture.ipynb))
    Dimensionality Reduction ([PCA.ipynb](PCA.ipynb))
